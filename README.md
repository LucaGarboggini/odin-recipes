# Odin Recipes 🍽️

This project is part of **The Odin Project** Foundations course.

It is a simple recipe website built using only **HTML**. The goal of this project is to practice creating web pages, working with links, images, lists, and organizing content using semantic HTML.

## Recipes Included

- 🍫 Brigadeiro
- 🍌 Banana Bread
- 🍕 Pizza

Each recipe contains:
- A brief description
- An ingredients list
- Step-by-step cooking instructions
- An image of the finished dish

## Skills Demonstrated

By completing this project, I practiced:

- Creating HTML documents
- Using headings, paragraphs, and lists
- Adding images with descriptive alt text
- Creating relative links between pages
- Organizing files into folders
- Using Git and GitHub for version control
- Writing meaningful commit messages

## Technologies Used

- HTML5
- Git
- GitHub
